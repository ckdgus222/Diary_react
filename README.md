* Vite
* Locel Storage


# :sunny: 주요 기능

* 그날에 운동한 기록을 남길수 있습니다.

* 월별로 기록했던 운동 기록 보기 
* 그날에 맞는 부위와 추천 스트레칭 영상 필터링
